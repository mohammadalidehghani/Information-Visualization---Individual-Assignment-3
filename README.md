# Individual Assignment 3 – D3.js Visualization  
Information Visualization – TU Wien

This project contains an interactive grouped bar chart created using D3.js.  
The visualization compares smoking prevalence and drug experimentation rates across different age groups in the year 2024.

## Description
The visualization uses a synthetic dataset containing around 10,000 records, focusing on youth smoking, drug experimentation, and related demographic and social factors.  
I selected a grouped bar chart to enable easy comparison between two key behavioral variables across multiple age groups.
